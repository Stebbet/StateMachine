Finite State Machines

A finite state machine is a mathematical model of computation that describes the behaviour of a system using states, transitions, and actions.
  • States are drawn as circles, with text inside for their name
  • Transitions are drawn as arrows
  • Actions are indicated as the labels on top of the transitions

Discrete Finite Automata

A discrete finite automata (DFA), used as the Default Machine on this site, is a type of finite state machine that processes a given input and accepts or rejects the input based on the given states and transition. DFA are particularly useful in pattern recognition systems, examples include language recognition, compiler design, and cybersecurity. They have a binary output, True or False, and accepting states are indicated as a double circle.
  • To create an accepting state, double-click on an existing state

Mealy Machines

A Mealy machine is a type of finite state machine whose output depends on the current state, and the current output. The transitions, are represented as X|Y, where X is the input and Y is its respective output. It does not have an accepting state.
  • Start designing a Mealy machine by opening the sidebar and toggling it in the settings tab
  • Change the output by clicking on the output part of the transition label

Moore Machines

A Moore machine is a type of finite state machine whose output is determined by just the current state, So the outputs are represented inside the state, underneath the state name. It does not have an accepting state.
  • Start designing a Moore machine by opening the sidebar and toggling it in the settings tab
  • Change the output by clicking on the output part of the transition label

How do I draw a machine?

  • Double-click on the canvas to create a new state. Or use the "new state" tool from the toolbar and single click
  • Toggle an initial state by clicking the tab shown underneath the state
  • Press shift then click and drag from a state to create a new transition. Or use the "new transition" tool from the toolbar then click and drag
  • Select a state by clicking on it. When a state is selected, you can change its name by typing
  • Select a transition by clicking on it. When a transition is selected, an orange anchor will show, you can the shape of the transition shape by dragging the anchor. Change the label on the transition by selecting it and typing
  • Straighten a transition by double-clicking it
  • Undo changes by clicking the undo button on the toolbar
  • Redo changes by clicking the redo button on the toolbar

Text

  • Click on a state or node to change the text inside of it. For Mealy and Moore machines, you must click on the text you want to change
  • Add greek letters by typing \[letter_name] e.g., \alpha, \beta

How do I simulate a machine?

  • Open the sidebar and click on the simulator tab to access the simulator.
  • Input comma separated values into the input area. e.g., 1,0,1,0
  • Run the simulation by either stepping through the inputs or processing all of them at once
  • The results will be shown in the console and the output will be shown in the output tab. Outputs will also be shown on the drawn machine and the current state will be visualised
  • Copy the output by clicking the copy button next to the output tab. Copy the console output by clicking inside the console

Saving

  • Save a machine by clicking the save button at the top of the page
  • You must be logged in to save a machine
  • Click the FSM logo to navigate to your dashboard. Here you can access your saved machines
  • Click your saved machine in the dashboard to access it, then you can update it
  • Delete your saved machine by clicking the delete button on the respective machine in your dashboard
